# note-app
Simple and secure note taking webapp built in Koajs, Postgresql &amp; Quasar.
### server env
```
PGHOST='localhost'
PGUSER=username
PGDATABASE=noteapp
PGPASSWORD=password
PGPORT=5432
PORT=8888
SESSIONKEY=key
```
